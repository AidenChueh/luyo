# luyo

個人旅遊規劃 App 的設計與開發專案。整合行程規劃、地圖、記帳、日誌、清單、AI 輔助等功能，目標是一站式的旅遊籌備與記錄工具。

## 用途

- 規劃多趟旅程，集中管理封面、日期、預算、狀態
- 逐日行程時間軸、地圖路線、多幣別記帳、旅遊日誌
- 行前準備（待辦 / 打包 / 購物清單）、機票住宿、地點收藏
- AI 行程規劃、預算分析、旅遊回顧
- 行後自動生成旅程總結與可分享報告卡

## 目前狀態

- 設計階段。主要交付物：`design-outline.html`（標註式設計大綱第一版草稿）

## 設計基調

- Apple HIG 原則：直覺導航、無障礙
- Airbnb + Google Travel + Notion 的視覺混搭
- 暖色系、16px 圓角、柔和陰影、Mobile-first 響應式

## 檔案

- `design-outline.html` — 完整功能與設計標註式大綱（可瀏覽器開啟）
- `CLAUDE.md` — 專案脈絡與變更紀錄
- `CHANGELOG.zh.md` — 中文變更紀錄

## 使用

直接用瀏覽器開啟 `design-outline.html` 閱讀設計草稿。
